# RetailPOS.API — Point of Sale Backend System

A production-ready REST API for retail operations built with **ASP.NET Core**, **Entity Framework Core**, and **SQL Server**. Implements secure authentication, role-based authorization, and atomic transaction processing for inventory and sales management.

## 🛠️ Tech Stack
- **Framework:** .NET 10 / ASP.NET Core Web API
- **Language:** C#
- **Database:** SQL Server + Entity Framework Core
- **Authentication:** JWT (JSON Web Tokens) + BCrypt password hashing
- **Documentation:** Swagger/OpenAPI
- **Architecture:** Clean Architecture, DTOs, Service Layer, Dependency Injection, Global Exception Middleware

## ✨ Key Features
- ✅ **Complete REST API** for retail operations (Products, Sales, Users)
- 🔐 **JWT Authentication** with Role-Based Access Control (Admin/Manager/Cashier)
- 📦 **Inventory Management** with automatic stock tracking
- 🛒 **Sales Transactions** with atomic updates and audit logging
- 📊 **Low Stock Alerts** and category filtering
- 🛡️ **Global Exception Handling** middleware for clean error responses
- 📖 **Interactive API Documentation** via Swagger UI
- 🔒 **Secure Password Storage** using BCrypt hashing
